先看毫秒级的ITCH数据
ITCH数据通常指纳斯达克交易所提供的ITCH协议数据，这是一种低延迟市场数据馈送系统，用于传输证券订单簿更新、交易执行等信息

B buy：增加买单
S sell：增加卖单
E execute：部分执行未完成的订单 价格大于0
C cancel：部分取消未完成的订单 价格等于0
F finish：完全执行未完成的订单 剩余股份等于0 价格等于0
D delete：完全删除未完成的订单 剩余股份等于0 价格等于0
	C和D的的订单号与之前配对的B/S一致，从而能让人完成配对
	还需要价格的话，也得从订单号一致的，之前的B/S中读取
X cross：完全删除未完成的订单
T 执行未显示的订单（具体呢？？？）

从这些数据，可以重建在任何时间点完整的订单簿LOB（MO不会在这里记录）

价格过程就是投资者之间相互互动的结果，彼此买卖，调整头寸


 
每日平均成交量表

平均每日交易数 记为N
	除以 6.5h*3600 得到每秒钟交易次数，因为美国开市时间为 9:30-16:00
	这直接衡量了市场在每秒钟内的“跳动”次数，是衡量市场实时活跃节奏的最直观指标。
纳斯达克平均每日总交易股份美元价值 记为V($)
	平均每笔交易金额：V($) / N，金额能看出是散户（金额小）还是机构（金额大）主导；

纳斯达克平均每日总交易股份交易数 记为V(Q)
	平均每笔交易股数：V(Q) / N

	V($) / V(Q) 计算出的是：按成交量加权的平均股价
	近似反映当年的平均股价：如果交易分布在全年分布比较平均，应该是的
	当然，极端一点，假如前364天没有交易，全都在最后一天交易，那么这个值反映的就是最后一天的平均股价，而不是全年的
	实际情况：真实的交易虽然不会像你的极端案例那样，但常呈现 “不均匀分布” 。例如，在财报发布日、重大新闻事件日或市场剧烈波动时，交易量会急剧放大，这几天的价格对V($)/V(Q)的影响就会远远大于那些交易清淡的日常。

	还是应该先计算每日的平均股价（日成交金额/日成交股数），再对这些日度价格进行简单算术平均

	苹果 V($) / V(Q) = 1505175/3208 = 469.19
	2012年，苹果公司是全球市值最高的公司之一，其股价在年初至9月期间表现强劲，股价从约400美元上涨至700美元。然而，随着市场对iPhone 5的预期逐渐消化，以及对公司未来增长潜力的质疑，苹果股价在第四季度出现了显著回调，年底时股价回落至约530美元，回调幅度超过20%。
	2013年苹果股票当年涨幅5.42%，年末股价561.02美元，数量级基本相符
	2013年12月31日苹果股价4903美元（后复权计算）

	假设你的苹果树现在有100个苹果，认为就是100股，一股对应一个苹果
	假如市价10块钱买1个苹果，那么10块钱也就对应一股，股价是10块钱
	后来，公司决定给股东“送股”，比如每持有10股送10股（相当于把你的苹果树劈成两半，每半树上苹果数量暂时不变）。
	送股后，你持有的股票数量变成了200股，但因为苹果总数没变（还是100个），每股对应的苹果就少了一半，只有半个苹果
	市价还是10块钱买1个苹果，但现在10块钱对应2股，所以股价会立刻从10块“跳”到5块。
	
	1、不复权：在股价走势图上，你会看到一个巨大的向下“缺口”（从10块直接掉到5块）。这个缺口只是因为公司送了股（或分了红），并不是公司真的突然贬值了那么多。不复权的图，中间会有很多这种“断裂”的缺口，看起来不连贯。

	2、“后复权” 价格。它的计算逻辑是：以当前的最新股价为基准，把历史上所有的分红、拆股等都“加回去”。

	后复权会想：“虽然现在股价是5块，
	但如果我们把后来送的那10股（相当于多出来的那半棵树）的价值加回去，把2新股拼回1旧股，1旧股应该值多少钱？还是10块
	此外，它会把送股、分红等带来的价值增长，都累计计算到当前的价格上。所以，在复权后的图上，现在的价格至少仍是10块，或者更高

	1、不复权：让整个k线图以当天的一股为单位而呈现，这可能造成k线突变
	2014年6月拆股前，一股价格467（假如2013年底的这个价格在半年内未动），拆股后，一股价格467/7约等于67，k线图出现断崖突变

	2、后复权：让整个k线图固定以开盘第一天的一股为单位而呈现
	这个数字告诉你：如果在开盘第一天买入当时的1股苹果并持有至今，并且把所有分红再投资，那么当初的那1股“等价”于今天的多少钱
	将k线图拆股后的k线都乘以7，使得可以与拆股前的k线拼接上。当然因为要把历史上的分红加上，所以k线图拆股后还是会出现突变

	3、前复权：让整个k线图固定以今天的一股为单位而呈现
	这个数字告诉你：如果在开盘第一天买入今天的1股苹果并持有至今，并且把所有分红都扣除，那么今天的1股苹果“等价”于当时的多少钱
	将k线图拆股前的k线都除以7，使得可以与拆股后的k线拼接上。当然因为要把未来的分红都扣除，所以k线图拆股后还是会出现突变

	2013年苹果的实际交易价格约469美元，回溯其历史中的拆股
	1987年6月：1股拆2股
	2000年6月：1股拆2股
	2005年2月：1股拆2股
	所以后复权计算中，系统会将2013年之前的所有历史价格，都乘以8，以与今天的股价保持连续。
	那么就应该是469*8=3752，再加上历史上的分红，可能就是4903了

	之后还有一次大拆股：2014年6月，苹果进行了1股拆7股。这意味着，在拆股前一天持有1股苹果的人，在拆股后变成了持有7股。拆股当日的股价会相应地除以7（例如从约650美元变为约93美元）。
	在后复权计算中，系统会将2014年拆股之前的所有历史价格，都乘以7，以与今天的股价保持连续。

全市场的平均每日总交易股份交易数 记为总V(Q)

	纳斯达克市场份额（按股数）：V(Q) / 总V(Q)

	这个比率衡量了该股票在纳斯达克交易所内的交易集中度。比率越高，说明该股票越能吸引该交易所的流动性。

流通股数量（Shares Outstanding），即公司已发行的、可在二级市场自由交易的股票总数，用于计算流通市值（流通股数量 × 当前股价）记为ShrOut

	日换手率：[总V(Q) / ShrOut]

	它表示每天有多少比例的流通股被转手

股票换手率 turnover：一年中总成交股份数 / 已发行股份数

	这个是年换手率，跟日换手率接近

假如N很高，但是 V($) / N 和 V(Q) / N 很小，则是散户（金额小）主导

	且说明 V(Q) 中等
	又假如 ShrOut 很高，则日换手率很低，说明资金深度不足，可能是低价大盘股。
